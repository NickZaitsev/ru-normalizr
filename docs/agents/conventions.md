# Output conventions

- Measure-head constructions such as `–і–Є–∞–Љ–µ—В—А–Њ–Љ/–≤—Л—Б–Њ—В–Њ–є/–Њ–±—К—С–Љ–Њ–Љ N` keep the
  numeral in the nominative; the head noun does not govern the measurement.
- Ranges with `—Б вА¶ –њ–Њ вА¶` end in singular `–≥–Њ–і` / `–≤–µ–Ї`; `–і–Њ вА¶ –≤–µ–Ї–∞` is also
  singular.
- Byte-family units use ordinary Russian counting forms consistently:
  `1 –±–∞–є—В`, `2вАУ4 –±–∞–є—В–∞`, `5 –±–∞–є—В–Њ–≤`, including prefixed units.
- References governed by `—Б–Љ.` / `—Б–Љ–Њ—В—А–Є` use an accusative reference noun and
  a cardinal label: `—Б–Љ–Њ—В—А–Є —А–Є—Б—Г–љ–Њ–Ї –і–≤–∞, —В–∞–±–ї–Є—Ж—Г —В—А–Є, —Б—В—А–∞–љ–Є—Ж—Г —З–µ—В—Л—А–µ`.
- Initial expansion never injects commas that are absent from the source.
- Latin `A` is read as `—Н–є`; Cyrillic initials use the actual letter name
  (`–Х. вЖТ –µ`), without vowel doubling or synthetic punctuation.
